## Features
- Animated particle network background
- Terminal typewriter hero animation
- Scroll-reveal animations
- Fully responsive (mobile-first)
- Dark cybersecurity theme
- Zero dependencies — pure HTML, CSS, JS

## Deploy to GitHub Pages

```bash
# 1. Create a new repo on GitHub named: portfolio
# 2. Run these commands:

git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/animhassen/portfolio.git
git push -u origin main

# 3. Go to repo Settings → Pages → Source: Deploy from branch → main → / (root) → Save
# 4. Your site will be live at: https://animhassen.github.io/portfolio
```

## Local Preview

Just open `index.html` in any browser — no build step needed.
